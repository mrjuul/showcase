06 Rules Files 5 always loaded

FilePurpose
email-routing.md Strict account mapping per org domain — never cross accounts
containment.md No filesystem/cmd/push in same turn as untrusted content
security.md No hardcoded secrets, confirm before delete / email / push
mcp-hygiene.md State tool + reason, max 3 chained MCP calls without confirmation
source-availability.md Verify sources reachable, never produce silently partial results

07 Workspace Domains

tech/

  • allow SSH, Windows-MCP, Bash
  • deny Google Workspace, Gmail

orgs/

  • allow GWorkspace, Dinero, Asana
  • deny SSH, nmap, Win-MCP

personal/

  • allow Personal Gmail only
  • deny SSH, nmap, org MCPs

The Problem

LLMs Forget Everything

Large language models are stateless. Every request is a blank slate. They can't remember your name, your preferences, or what you discussed five minutes ago. Agentic memory solves this — but the landscape of solutions is fragmented, fast-moving, and full of tradeoffs. This survey maps the terrain.

Patterns

Cross-Cutting Architecture Patterns

Six patterns appear repeatedly across frameworks — the building blocks of agentic memory.

Scored Retrieval

Zep, Mem0, LangGraph

Multi-factor scoring (recency, importance, relevance, access frequency) to select which memories enter the prompt. Implementations differ: Zep uses bi-temporal validity, Mem0 uses cosine similarity + graph traversal.

Virtual Context / Sliding Window

MemGPT/Letta, LangGraph, Anthropic

Fixed context window as "RAM" with external storage as "disk." The agent or framework manages paging. MemGPT's insight: let the LLM itself be the memory manager.

Knowledge Graph Extraction

Cognee, Zep/Graphiti, Mem0

LLM extracts entities and relationships from unstructured text, builds a graph. Enables relational queries that pure vector search cannot answer.

Reflection / Consolidation

Letta, Ng's patterns, Weng's Reflexion

Periodic synthesis of episodic memories into semantic knowledge. Ranges from per-turn recursive summarization (Letta) to nightly batch consolidation.

Memory as Filesystem

Anthropic, Letta (2026), LangChain Deep Agents

Memory lives as files (markdown, JSON) that agents read/write via standard file operations. Multiple teams independently converging on this pattern — models are "good at using filesystems."

Client-Side Memory

Anthropic, LangGraph, self-hosted options

Memory infrastructure runs in the developer's environment, not the provider's. Avoids lock-in, enables custom persistence, full data ownership.

Trends

Where It's Heading

Memory as Filesystem

Letta, Anthropic, and LangChain are all converging on file-based memory over specialized tools. Letta's own benchmarks show filesystem agents outperform specialized memory abstractions.

Temporal Reasoning Gap

Zep's bi-temporal model is the gold standard for tracking how facts change over time. Others lag significantly — a 15-point gap on LongMemEval between Zep and Mem0.

Self-Improving Memory

Cognee's memify, MemGPT's self-editing, and biological consolidation patterns show memory evolving from static storage to adaptive systems that learn from usage.

Benchmark Maturation

MemoryArena (2026) reveals that models scoring near-perfect on LOCOMO plummet to 40-60% on interdependent multi-session tasks. The real frontier is multi-session, not single-turn.
